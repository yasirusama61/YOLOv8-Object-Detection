# YOLOv8 Metrics

## Overview:
YOLOv8 was optimized with advanced augmentation and WeightedRandomSampler for detecting thoracic abnormalities.

### **Performance Metrics**
- **Precision**: ~70%
- **Recall**: ~65%
- **mAP@50**: ~0.497
- **mAP@50-95**: ~0.271

### **Detailed Observations**
- **Strengths**:
  - Faster inference time suitable for real-time applications.
- **Challenges**:
  - Struggled with minority classes like Mass and Atelectasis, despite augmentation.
